# sticker-maker
<h2>Summery</h2>
The application allows you to chose the paper size and the images you want to make as a stickers collection then it converts them to svgs for better printing result and then you can download it.
<h2>Built with</h2>
<ul>
  <li>ReactJS</li>
  <li>Chakra UI</li>
  <li>redux toolkit</li>
  <li>NodeJS</li>
</ul>
